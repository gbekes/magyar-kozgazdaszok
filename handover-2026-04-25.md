## One-paragraph project summary

**Magyar Közgazdászok** — static catalogue of peer-reviewed research,
policy work, and press output by Hungarian economists, framed for
Hungarian policymakers. HTML + vanilla JS + CSS, JSON data layer,
GitHub Pages deploy on push to `main`. The catalogue now distinguishes
three content categories: **Research** (`data/papers/`), **Policy**
(`data/policy/` — long-form institutional reports / book chapters /
WPs in CEPR-NBER), **Press** (`data/press/` — short-form: op-eds,
columns including VoxEU and Index *defacto*, interviews, podcasts,
blog posts). Author pages render all three sections; counter chips
show "X research · Y policy · Z press". Authors index also shows the
counts inline. Bilingual everywhere; HU titles populated for all 87
Hungarian-origin press items.

## Format / repo conventions to remember

- `[x]` in the seed markdown means **drop** (editor-flipped from the
  default convention). Blank brackets keep.
- Bilingual title rule: `title` is canonical English; `title_hu` is
  Hungarian. Renderer prefers the page-language version.
- `outlet_kind` ∈ `{report, chapter, working_paper}` — but per the
  2026-04-25 rule, no `working_paper` policy items unless they're
  recent NBER/CEPR pre-publication.
- VoxEU + Index *defacto* count as **Press** (short-form).
- Bruegel reports/Blueprints/blog posts: report → Policy; blog/
  newsletter/podcast → Press.
- Don't push commits unless instructed (editor's repeated
  preference). Auto-push is enabled for the magyar-kozgazdaszok
  repo per the saved memory note.
