Belépő folyóiratok

A katalógusba akkor kerül be egy közgazdász, ha legalább egy publikációja megjelent az alábbi folyóiratok valamelyikében. A bekerülés után minden tanulmánya és friss műhelytanulmánya bekerülhet a katalógusba.

Nagyjából ez a D1 — vezető általános érdekű folyóiratok — és a legjobb területi folyóiratok: munkaügyi, kereskedelem, pénzügy, makroökonómia, IO, közgazdaságtan, fejlődés, ökonometria, elmélet, menedzsment, városi, egészségügy, környezet, gazdaságtörténet. Körülbelül a közgazdaságtan legjobb 50 folyóirata, kiegészítve a vezető pénzügyi és menedzsment folyóiratokkal.

Minden ilyen lista önkényes. Javaslatokat GitHub issue vagy PR formájában várunk.
