# magyar-kozgazdaszok — Evidence for Hungary

A searchable, policy-facing catalogue of research by top Hungarian economists.
Static site; no framework, no build step for the pages themselves. See the live
site at (domain TBD).

## Quick start (local)

```bash
python build.py            # regenerates data/index.json
python -m http.server 8080 # serves the site
```

Open <http://localhost:8080/>. Hungarian mirror at `/hu/`.

## What's in this repo

## Adding a paper

1. Copy `sample-paper.json` to `data/papers/<slug>.json`.
2. Edit the fields. Slug format: `<lead-surname>[-<second-surname>|-et-al]-<year>-<short-outlet>`.
3. `python build.py`
4. Refresh the browser.

## Adding an author

1. Either add an entry to `authors-seed.json` and run `python build.py` (which
   creates `data/authors/<slug>.json` for any new seed entries), or create the
   file directly.
2. `python build.py`

## Ingesting at scale

```bash
python -m pip install -r scripts/requirements.txt
export ANTHROPIC_API_KEY=sk-ant-...   # or set via your shell
python scripts/ingest.py run <author_id> --max-works 10
python build.py
```

Full workflow and safety guarantees are described in [BUILD.md](BUILD.md).

## Deploy

Any static host.

### GitHub Pages (one-click)

This repo includes `.github/workflows/pages.yml` which publishes the whole
site on every push to `main`. Enable Pages in the repo settings →
Pages → Source = "GitHub Actions".

**Custom domain:** once you've bought a domain, add a file at the repo root
named `CNAME` containing just the domain (e.g., `evidence-hu.org`), push, and
configure the DNS at the registrar as GitHub Pages instructs (usually an A or
CNAME record pointing at `username.github.io`).

### Vercel / Netlify / Cloudflare Pages

Point at the repo root, no build command, output directory `.`. `build.py` can
be run as a build step if you want `data/index.json` refreshed on deploy:

```
python build.py
```

## Review and editorial

All AI-generated content is marked `review_status: ai-drafted`. After a human
editor checks and edits, flip to `human-reviewed`. If the author signs off, flip
to `author-approved`.
