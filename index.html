How to use this site →

This page collects papers from Hungarian economists with at least one top publication — defined as a paper in a top general-interest journal (D1) or a top field journal (roughly A and A–). In practice, this means roughly the top 50 economics journals plus top finance and management outlets.

See the exact journal list →

Any such list is arbitrary. File an issue or PR at the GitHub repo to suggest changes.

The content of the website is under development, some text is not yet human verified. If you are a Hungarian economist: please suggest names (self-promotion welcome) and check your own info and the summaries of your papers and help improve the list. Thanks!
